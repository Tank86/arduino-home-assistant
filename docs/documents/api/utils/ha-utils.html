static uint16_t numberToStr(char *dst, Number value, const uint8_t precision = 0)¶

Converts the given number to the string. It can be integer or floating point number. In a case of float the input value needs to be multiped by pow(10, precision).

Note

The dst size should be calculated using HAUtils::calculateNumberSize method plus 1 extra byte for the null terminator.

Parameters

  • dst – Destination where the number will be saved. The null terminator is not added at the end.

  • value – Number to convert.

  • precision – Number of decimal digits in the number.

Returns

The number of written characters.
